# Process Scheduler Simulator


## Overview

This project simulates common CPU scheduling algorithms used in operating systems.

The simulator implements Round Robin (RR), Shortest Job First (SJF), and Shortest Remaining Time First (SRTF) scheduling strategies while tracking process execution, waiting time, turnaround time, and preemption behavior.


## Implemented Algorithms

- Round Robin (RR)
- Shortest Job First (SJF)
- Shortest Remaining Time First (SRTF)


## Features

- Process scheduling simulation
- Preemptive and non-preemptive scheduling
- Waiting time calculations
- Turnaround time calculations
- Gantt-style execution output


## Technologies

- Python
